This is a random walk game where you roll a die and the following rules apply:

die roll <= 2: you go down a step
2 < die roll <= 5: you go up a step
die_roll = 6: you go up die_roll steps

you also have a 0.1% chance of falling down to step 0 -- awkward!

this piece of code shows a histogram of what steps you get to, then a line graph of the random walks (where you can see the clumsiness of falling to step 0)
and the chance of getting past step 100. wahoo!
